### To start the project you need

Clone this repo
Install react-native cli
Real Device or Emulators

### Run project

First command
`yarn install` or `npm install`

Two command
`react-native link`
`react-native run-ios` / `react-native run-android`
Or
Separate command
`react-native start`
`react-native link`
`react-native run-ios` / `react-native run-android`
